Diesel_API

This web application provides a wide range of information about diesel and costs associated with it. User can follow current information from stock trading (API). Moreover User can calculate travel cost, when he uses diesel car. User completes form and gets information about travel cost between 2 cities in Poland. To calculation user can choose from databse or add his car. User can alco compare his car with the lowest fuel consumption diesel car.

Pregnant_Pills

This app is for pregnant women to keep information about pills, which they took during pregnancy. On home page user can register a new account or continue as guest. When user is logged in, it is possible to create a list with pills. User can add a new pill or choose from database. Pregnancy week is also required. After that user gets a list with pills (with dates and type of pills), which is avaiable to download this list as PDF file.
